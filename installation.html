How it Works¶

We built a simulation framework in order to perform power flow study in the U.S. electrical grid. The framework and the available features are presented in the README of the PowerSimData package. This is the first release and, unfortunately, you won’t be able to use this package as the researchers of our team do unless you have the following set up:

  • A server with a Unix-like operating system - the engine must be installed on the server and the client-server communication is done over ssh

  • A Gurobi license - this is the mathematical solver that our engine use to find power flow solutions to the DC Optimal Power Flow (DCOPF) problems

It is also possible to run the simulation on your laptop/desktop without using PowerSimData package. A Gurobi license is still needed but a set of input data to run a simulation is available for download. The two types of installation are presented in the here.

Note that more user friendly setups will be available in the next releases. Docker will be used to package up the code and all its dependencies simplifying installation, development and deployment. Also, we will rely on cloud computing to run scenarios.
