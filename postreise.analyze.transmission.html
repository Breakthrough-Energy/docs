postreise.analyze.transmission.utilization module¶

postreise.analyze.transmission.utilization.generate_cong_stats(pf, grid_branch, util=None, thresh=None)[source]¶

Generates congestion/utilization statistics from powerflow data (WECC congestion reports’ analyses are the inspiration for these analyses and are the source of the default parameters).

Parameters

  • pf (pandas.DataFrame) – power flow data frame

  • grid_branch (pandas.DataFrame) – grid.branch branch info

  • util (list) – utilization (float) flag level 1, 2, 3. Default values are values used by WECC: 0.75, 0.9, 99.

  • thresh (list) – threshold for proportion time, for flag level 1, 2, 3. Default values are values used by WECC: 0.5, 0.2, 0.05.

Returns

(pandas.DataFrame) – congestion statistics. ‘per_util1’, ‘per_util2’, ‘per_util3’, ‘u1flag’, ‘u2flag’, ‘u3flag’, ‘sumflag’, ‘bind’, ‘risk’.
