area_to_loadzone(*args, **kwargs)[source]¶

Map the query area to a list of loadzones, using the known grid model.

powersimdata.network.model.area_to_loadzone(grid_model, area, area_type=None, mappings=None)[source]¶

Map the query area to a list of loadzones.

Parameters

  • grid_model (str) – the grid model to use to look up constants for mapping.

  • area (str) – one of: loadzone, state, state abbreviation, interconnect, ‘all’.

  • area_type (str) – one of: ‘loadzone’, ‘state’, ‘state_abbr’, ‘interconnect’.

  • mappings (iterable) – a set of strings, representing area types to use to map. If None, all mappings are tried.

Returns

(set) – set of loadzone names associated with the query area.

Raises

  • TypeError – if area is not None or str.

  • ValueError – if area is invalid or the combination of area and area_type is invalid.

  • KeyError – if a mapping is provided which isn’t present for a grid_model.

Note

if area_type is not specified, the function will check the area in the order of ‘state’, ‘loadzone’, ‘state abbreviation’, ‘interconnect’ and ‘all’.
